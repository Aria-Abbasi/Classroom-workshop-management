## How to start

1. Clone the repo
   ```sh
   git clone https://github.com/Aria-Abbasi/Classroom-workshop-management.git
   ```
2. Install required dependencies
   ```sh
   pip install -r requirements.txt
   ```
4. Start development server
   ```js
   python3 manage.py runserver
   ```
